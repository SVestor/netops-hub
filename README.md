# 📡 NetworkHub: Automated GNS3 Deployment in AWS
NetworkHub is an infrastructure automation project designed for deploying a GNS3 Server in AWS Cloud using Terraform. 
It includes configurations for seamless integration with your local network, automated networking setups, and essential debugging tools.

---

## ⚙️ Features

- Terraform Automation – Fully scripted deployment of GNS3 in AWS
- Seamless Connectivity – Route configuration for local and cloud networks
- Network Debugging – Essential commands for Cisco, Linux, and Windows
- Traffic Management – iptables, UFW, and routing best practices

---

## ⚡ Quick Start
Refer to the provided configuration files for setup details, routing rules, and debugging tools.
